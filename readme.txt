<プロジェクト概要>
スマートフォン向けの2Dアクションゲームです。
ビルドデータの動作環境：Windows10,iOS(iphone11の14.8で確認)
実行ファイルは"Build"内にあります。

<詳細>
　Unity,C#で開発された、スマートフォン向け2D横スクロールアクションゲームです。開発人数は一人、大学の授業が忙しい時期や長期休業期間が含まれるため目安ですが、ここまでの制作期間は5か月程です。
　このゲームは、「スマホで思い通りに動かせる横スクロール」を目指して作られています。重力の概念があるスマホゲームは、操作のしやすさと動きの多彩さ、いずれかが犠牲になりがちです。そこで見下ろし型のアクションゲームを参考に、画面上どこでもスワイプで移動、フリックでダッシュ、タップで攻撃、というように一本指で多岐に渡るアクションを使い分けられるように設計しました。
　今後も改良を進めていく予定で、難易度や細かい操作感の調整、エフェクトの修正を行ったのち、残り6つほどのステージ、シナリオイベント、コンフィグ機能、広告表示などの追加を考えています。順調に進めば今年度中にはリリースできると考えています。

<ステージ画面の操作> 
スワイプ（ドラッグ）：左右移動
フリック（素早くドラッグして離す）：ダッシュ・ジャンプ　空中で一回まで追加ダッシュ可能
タップ（クリック）：攻撃
左上の数値が5Tb以上の時、長押し：溜め
溜め中その場で離す：特殊攻撃N
溜め中スワイプ（ドラッグ）して離す：特殊攻撃上・横・下
